## 📖 Описание

Проект представляет собой комплексный курс из **21 лабораторной работы**, охватывающих ключевые технологии современного **NLP**, **компьютерного зрения**, **мультимодальных систем**, **прикладного ИИ** и **специализированных техник**. Каждая работа построена как самостоятельное Jupyter-ноутбук с подробными объяснениями, примерами кода и практическими заданиями.

## 📖 Подробное описание лабораторных

### 🔹 Лабораторная 1: Введение в LLM

**Цель:** Знакомство с предобученными языковыми моделями и экосистемой Hugging Face.

**Что вы изучите:**
- Работа с `Hugging Face pipeline()` для упрощения взаимодействия с моделями
- Генерация текста с различными параметрами (temperature, top-k, top-p)
- Классификация тональности текста (sentiment analysis)
- Построение Question-Answering систем

**Результат:** Готовые скрипты для анализа текста и генерации ответов.

---

### 🔹 Лабораторная 2: Fine-tuning

**Цель:** Освоение техник эффективной дообучения больших языковых моделей.

**Что вы изучите:**
- Transfer Learning в NLP и его преимущества
- PEFT (Parameter-Efficient Fine-Tuning) и LoRA адаптеры
- 4-bit квантование для экономии памяти
- Дообучение модели на датасете инструкций (Dolly-15k)

**Результат:** Дообученная модель для генерации инструкций.

---

### 🔹 Лабораторная 3: RAG (Retrieval-Augmented Generation)

**Цель:** Построение системы ответов на вопросы с использованием внешней базы знаний.

**Что вы изучите:**
- Борьба с галлюцинациями LLM через внешний контекст
- Создание и использование эмбеддингов
- Векторный поиск с помощью FAISS
- Генерация ответов с учётом найденного контекста

**Результат:** RAG-система для ответов на вопросы по документам.

---

### 🔹 Лабораторная 4: Векторные базы данных

**Цель:** Изучение современных векторных баз данных для хранения и поиска эмбеддингов.

**Что вы изучите:**
- ChromaDB для хранения и управления эмбеддингами
- Семантический поиск по текстовым данным
- Фильтрация результатов по метаданным
- Гибридный поиск (векторный + ключевые слова)

**Результат:** Система семантического поиска с метаданными.

---

### 🔹 Лабораторная 5: Мультимодальные LLM

**Цель:** Работа с моделями, способными обрабатывать текст и изображения одновременно.

**Что вы изучите:**
- Архитектура Qwen2.5-VL и её особенности
- Visual Question Answering (VQA)
- Генерация описаний изображений (image captioning)
- Анализ графиков, диаграмм и визуальных данных

**Результат:** Мультимодальная система для анализа изображений.

---

### 🔹 Лабораторная 6: Vision-Language Models

**Цель:** Изучение контрастивного обучения и zero-shot классификации.

**Что вы изучите:**
- CLIP и контрастивное обучение
- Zero-shot классификация изображений без дообучения
- Поиск изображений по текстовому описанию
- Сравнение архитектур (ViT-B/32, ViT-L/14, RN50)

**Результат:** Система zero-shot классификации и поиска изображений.

---

### 🔹 Лабораторная 7: Разработка автономных AI-агентов

**Цель:** Создание интеллектуальных агентов, способных планировать и выполнять многошаговые задачи.

**Что вы изучите:**
- Архитектура AI-агентов и их компоненты
- ReAct паттерн (Reasoning + Acting)
- LangGraph для построения графов выполнения
- Интеграция с внешними инструментами и API

**Результат:** Автономный агент для решения сложных задач.

---

### 🔹 Лабораторная 8: Оптимизация и деплой LLM

**Цель:** Освоение методов ускорения инференса и развертывания моделей в продакшене.

**Что вы изучите:**
- Квантование моделей (4-bit, 8-bit)
- vLLM для высокоскоростного инференса
- Создание API через FastAPI
- Контейнеризация с Docker

**Результат:** Готовое к деплою решение с оптимизированным инференсом.

---

### 🔹 Лабораторная 9: Оценка качества RAG-систем

**Цель:** Научиться автоматически оценивать качество RAG-систем.

**Что вы изучите:**
- Метрики Faithfulness, Answer Relevance, Context Precision
- Фреймворк RAGAS для комплексной оценки
- DeepEval для тестирования генерации
- Построение датасетов для оценки

**Результат:** Система автоматической оценки качества RAG.

---

### 🔹 Лабораторная 10: Безопасность LLM

**Цель:** Изучение уязвимостей LLM и методов защиты.

**Что вы изучите:**
- Prompt injection атаки и защита
- Jailbreaking техники и контрмеры
- Guardrails для ограничения вывода
- Llama Guard и NeMo Guardrails

**Результат:** Защищённая LLM-система с детекцией атак.

---

### 🔹 Лабораторная 11: Локальные малые модели и дистилляция

**Цель:** Работа с компактными моделями для локального запуска.

**Что вы изучите:**
- Архитектуры SLM (TinyLlama, Phi-3, Gemma)
- Дистилляция знаний от больших моделей
- Сравнение качества и скорости
- Оптимизация для edge-устройств

**Результат:** Локально работающая модель с хорошим качеством.

---

### 🔹 Лабораторная 12: Function Calling и работа с инструментами

**Цель:** Обучение модели вызову внешних функций на основе запроса.

**Что вы изучите:**
- Function Calling API и формат JSON Schema
- Создание инструментов через @tool декоратор
- Интеграция с внешними API (погода, валюта, поиск)
- Обработка ошибок и логирование

**Результат:** LLM с возможностью вызова внешних функций.

---

### 🔹 Лабораторная 13: Продвинутый RAG (Hybrid Search + Re-ranking)

**Цель:** Улучшение качества поиска за счёт гибридных методов.

**Что вы изучите:**
- Комбинация семантического и ключевого поиска
- Cross-encoder модели для reranking
- Cohere Rerank и другие сервисы
- Оптимизация точности выдачи

**Результат:** RAG-система с улучшенным качеством поиска.

---

### 🔹 Лабораторная 14: Генерация изображений

**Цель:** Освоение диффузионных моделей для генерации изображений.

**Что вы изучите:**
- Stable Diffusion и архитектура диффузии
- Text-to-Image и Image-to-Image генерация
- ControlNet для контроля композиции
- LoRA для тонкой настройки стиля

**Результат:** Система генерации изображений по тексту.

---

### 🔹 Лабораторная 15: Speech-to-Text и Text-to-Speech

**Цель:** Создание голосового интерфейса для взаимодействия с ИИ.

**Что вы изучите:**
- Whisper для распознавания речи
- Silero TTS и XTTS для синтеза голоса
- Обработка аудио через torch_audio
- Построение голосового ассистента

**Результат:** Голосовой интерфейс для LLM.

---

### 🔹 Лабораторная 16: Оценка и тестирование промптов

**Цель:** Систематический подход к оптимизации промптов.

**Что вы изучите:**
- DSPy для программируемой оптимизации
- LangSmith для трассировки и отладки
- Создание датасетов для тестирования
- Метрики качества промптов

**Результат:** Оптимизированные промпты с измеримым качеством.

---

### 🔹 Лабораторная 17: RAG для структурированных данных (Text-to-SQL)

**Цель:** Создание систем, преобразующих естественный язык в SQL-запросы для работы с базами данных.

**Что вы изучите:**
- Архитектура Text-to-SQL систем
- Динамическое получение схемы таблиц через SQLDatabaseToolkit
- Генерация SQL-запросов естественным языком
- Выполнение запросов и форматирование ответов
- Обработка ошибок синтаксиса SQL

**Результат:** Система для получения данных из БД через естественный язык.

---

### 🔹 Лабораторная 18: Графы знаний (Knowledge Graphs)

**Цель:** Использование графовых структур для улучшения поиска и понимания связей между сущностями.

## 🛠️ Troubleshooting

### Частые проблемы и решения

| Проблема | Решение |
|----------|---------|
| **Нехватка памяти GPU** | Используйте `bitsandbytes` для 4/8-bit квантования |
| **Ошибки при загрузке модели** | Проверьте доступ к Hugging Face Hub, используйте `HF_TOKEN` |
| **Медленная работа без GPU** | Установите CUDA-версию PyTorch или используйте Google Colab |
| **Конфликты зависимостей** | Создайте чистое виртуальное окружение |

### Полезные команды

```bash
# Проверка версии PyTorch и CUDA
python -c "import torch; print(torch.__version__); print(torch.cuda.is_available())"

# Очистка кэша Hugging Face
huggingface-cli logout
rm -rf ~/.cache/huggingface
```
